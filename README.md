# 🐍 贪吃蛇游戏

一个功能丰富的现代化贪吃蛇游戏，使用HTML5、CSS3和JavaScript实现。

## ✨ 游戏特色

### 🎮 多种游戏模式
- **经典模式**: 简单、中等、困难三种难度
- **生存模式**: 随时间递增的挑战模式，障碍物逐渐增加

### 🎨 动态主题系统
- **季节主题**: 春夏秋冬自动切换
- **节日主题**: 春节、万圣节特殊主题
- **动态背景**: 随时间变化的背景动画
- **粒子效果**: 吃到食物时的爆炸粒子效果

### ⚡ 能力提升系统
- **得分翻倍**: 15秒内得分翻倍
- **特殊图标**: 闪电形状的能力提升道具
- **动画效果**: 脉冲、闪烁动画吸引注意

### 📊 独立统计系统
- **分难度记录**: 每个难度独立统计最高分
- **实时更新**: 切换难度时自动显示对应记录
- **本地存储**: 使用localStorage持久化保存

## 🎯 游戏玩法

### 基础操作
1. 点击"开始游戏"按钮开始
2. 使用方向键或WASD键控制蛇的移动方向
3. 吃到食物增加分数和蛇的长度
4. 避免撞到障碍物或自己的身体
5. 使用"暂停"按钮或按空格键暂停游戏

### 生存模式
- **渐进难度**: 每30秒自动升级
- **障碍物增加**: 基础10个 + (等级-1) × 3个
- **速度提升**: 每级增加0.1倍速度
- **等级显示**: 实时显示当前生存等级

### 能力提升道具
- **闪电图标**: 金色背景的白色闪电图标
- **动画效果**: 脉冲、闪烁、缩放动画
- **得分翻倍**: 15秒内所有得分翻倍
- **随机生成**: 吃到大食物后随机出现

## 🎮 控制说明

### 键盘控制
- **↑ / W**: 向上移动
- **↓ / S**: 向下移动
- **← / A**: 向左移动
- **→ / D**: 向右移动
- **空格键**: 暂停/继续游戏

### 触摸控制（移动设备）
- **向上滑动**: 向上移动
- **向下滑动**: 向下移动
- **向左滑动**: 向左移动
- **向右滑动**: 向右移动

## 📊 难度设置

### 经典模式
- **简单**: 8个障碍物，适合初学者
- **中等**: 15个障碍物，标准挑战
- **困难**: 25个障碍物，高难度挑战

### 生存模式
- **基础**: 10个障碍物，150ms基础速度
- **升级**: 每30秒增加3个障碍物，速度提升0.1倍
- **无限**: 理论上可以无限升级，挑战极限

## 🏆 最高分系统

### 独立统计
- **简单模式**: `snakeHighScore_easy`
- **中等模式**: `snakeHighScore_medium`
- **困难模式**: `snakeHighScore_hard`
- **生存模式**: `snakeHighScore_survival`

### 功能特点
- **实时更新**: 切换难度时自动显示对应最高分
- **本地存储**: 使用localStorage持久化保存
- **独立记录**: 每个难度完全独立的统计

## 🎨 视觉效果

### 粒子系统
- **爆炸效果**: 吃到食物时的粒子爆炸
- **主题色彩**: 根据当前主题使用对应颜色
- **物理模拟**: 重力、速度、生命周期

### 动画效果
- **背景动画**: 动态渐变背景
- **能力道具**: 脉冲、闪烁动画
- **界面交互**: 悬停、点击动画效果

## 🛠️ 技术实现

### 核心技术
- **HTML5 Canvas**: 游戏渲染和动画
- **CSS3**: 界面样式和动画效果
- **JavaScript ES6+**: 游戏逻辑和交互
- **localStorage**: 数据持久化存储

### 架构特点
- **模块化设计**: 功能分离，易于维护
- **响应式布局**: 适配不同屏幕尺寸
- **性能优化**: 高效的渲染和更新机制
- **跨平台兼容**: 支持桌面和移动设备

## 🚀 如何运行

1. 下载项目文件
2. 直接在浏览器中打开 `index.html`
3. 选择难度模式开始游戏
4. 享受丰富的游戏体验！
